# MovieShelf_backend

Springboot + Java + Oracle Cloud DB

Java Spring Boot, JPA, React, Oracle Cloud DB 기반 웹 프로젝트.

사용자들간에 영화 정보와 리뷰를 공유하는 웹 페이지 제작 프로젝트입니다.

<br>

## 구현 기능

- 소셜 로그인 (구글, 카카오)
- 회원 JPA를 이용한 CRUD 
- 영화 추천 알고리즘
